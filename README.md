# ChinaBot (Telegram + Python)

Учебный Telegram-бот для изучения китайского языка:
- поиск слова (китайский / русский / пиньинь),
- карточки пользователя,
- тест по карточкам (4 варианта),
- тест «подбери слово» (пропуск в предложении).

## 1) Что нужно установить

### Windows
1. Установите Python 3.11+ с официального сайта: https://www.python.org/downloads/windows/
2. При установке обязательно включите галочку **Add Python to PATH**.
3. Откройте новый PowerShell и проверьте:

```powershell
python --version
pip --version
```

## 2) Создание Telegram-бота

1. В Telegram откройте `@BotFather`.
2. Выполните `/newbot`.
3. Задайте имя и username бота.
4. Скопируйте токен вида `123456:ABC...`.

## 3) Настройка проекта

Из папки проекта (`C:\4sem\Цифровая аналитика\ChinaBot`):

```powershell
python -m venv .venv
.\.venv\Scripts\Activate.ps1
pip install -r requirements.txt
Copy-Item .env.example .env
```

Откройте `.env` и вставьте токен:

```dotenv
BOT_TOKEN=ВАШ_ТОКЕН_ОТ_BOTFATHER
DB_PATH=data/chinabot.sqlite3
```

## 4) Запуск

```powershell
.\.venv\Scripts\Activate.ps1
python bot.py
```

Если все корректно, бот начнет polling и будет отвечать в Telegram.

## 5) Структура проекта

- `bot.py` — точка входа, запуск aiogram.
- `app/config.py` — чтение настроек из `.env`.
- `app/db.py` — SQLite-хранилище карточек.
- `app/handlers/` — сценарии Telegram-команд.
- `app/services/dictionary_service.py` — поиск по словарю.
- `app/services/example_service.py` — примеры предложений.
- `app/services/quiz_service.py` — генерация тестов.
- `app/keyboards.py` — кнопки и меню.
- `data/` — база данных.

## 6) Какие открытые источники данных используются

1. **CC-CEDICT** через пакет `cepy-dict` (китайский → английский словарь с пиньинем).
2. **Tatoeba API** для примеров предложений.
3. `deep-translator` для перевода значений/примеров в русский.

Важно:
- Этот проект не требует платного API-ключа.
- Для переводов и примеров нужен интернет.
- Перевод на русский формируется автоматически, поэтому иногда может быть неидеальным.

## 7) Как пользоваться ботом

### «Найти слово»
- Введите китайское слово, пиньинь (латиницей) или русское слово.
- Бот вернет иероглифы, пиньинь, значения и 2-3 примера.

### «Мои карточки»
- Можно создать карточку, просмотреть по номеру, удалить.

### «Тест по карточкам»
- Нужно минимум 4 карточки.
- Вопрос: китайское слово + 4 варианта перевода.

### «Подбери слово»
- Нужно минимум 10 карточек.
- Выбор длины теста: 10 / 15 / 20 предложений.
- В конце: результат + разбор ошибок.

## 8) Команды в Telegram

- `/start` — запуск
- `/menu` — показать меню
- `/cancel` — отменить текущий сценарий

Также поддерживается кнопка/текст `Отменить`.
